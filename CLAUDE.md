# しんちゃんクイズ プロジェクト

## プロジェクト概要

クレヨンしんちゃんをテーマにしたクイズアプリ。

## Git 運用ルール

- コードを変更するたびに GitHub にプッシュすること
- リモートリポジトリ: https://github.com/amotoki7/20260522_shinchan_quiz.git
- コミットメッセージは変更内容を簡潔に日本語で書くこと
- プッシュ前に動作確認を行うこと

## 業務ルール

- ファイルの削除・上書き前には必ず確認を求めること
- フォルダ名・ファイル名の日付は「YYYYMMDD」形式にすること
- 作業が完了したら `Claude_Work/作業ログ/` フォルダに実施内容を追記すること
